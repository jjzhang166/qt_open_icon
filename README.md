# QtOpenUtils #

## 项目特色 ##
借鉴WEB前端开发中图标使用字体库的方式生成，把这种方法也运用到QT开发中，摒弃传统的贴图方式，让你在QT开发中从繁琐的图片维护中解放出来。以此为核心，再增加各种常用组件。

## 运行说明 ##
基于qt5.2.1编译,如果是其它版本qt库，可能某些类不支持或者有变化（详见文档），或者可能需要更改头文件，请自行检查解决。
